# Cytos Extensions

Premium browser extension for `cytos.io` that features a unified, highly aesthetic morphing portal design, nearby player tracking, and custom visual adjustments.

## Features

- **Unified Morphing Portal:** A clean and dynamic fullscreen hub that blurs the background and morphs seamlessly between navigation and setting views.
- **Active Players List:** Real-time list tracking nearby players. Includes quick actions to copy skin URLs, player nicknames, and spectate players directly.
- **Name Customizer:** Personalize name display styles, including custom colors, gradients, and RGB modes.
- **Visual Theme Panel:** Control client theme configurations, highlighting colors, and panel transparency scales.

## Installation & Build

1. Clone or download this repository:
   ```bash
   git clone https://github.com/Ulas-Ozcelik/cytos-extensions.git
   ```
2. Install build dependencies:
   ```bash
   npm install
   ```
3. Compile the extension bundle:
   ```bash
   npm run build
   ```
4. Load the extension in your browser:
   - Open Chrome and navigate to `chrome://extensions/`.
   - Enable **Developer mode** (top right switch).
   - Click **Load unpacked** and select this directory.
